# Public Game Archive

Completed games are publicly viewable — anyone can read their content without logging in.

## Accessing Archived Games

Navigate directly to a completed game's URL (e.g., via a shared link). No account required. The full game page loads in read-only mode.

Active and in-progress games still require authentication to view.

## What You Can See

As an unauthenticated visitor to a completed game, you have access to these tabs:

- **History** — All past phases with their posts, comments, and polls
- **People** — All characters and participants
- **Handouts** — All published handouts
- **Audience** — The audience view of action submissions and private messages
- **Game Info** — Genre, dates, and player count

There is no standalone Common Room tab — Common Room content from each phase is accessible through the **History** tab.

Content that was private during the game (unpublished action results, private messages between participants) remains visible only to authenticated participants with access.
