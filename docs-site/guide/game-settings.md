# Game Settings

When creating or editing a game, the GM configures a set of options that affect how the game runs. These settings can be changed at any time while the game is active using **Edit Game** from the game actions menu (⋮). Edit Game is not available after a game is completed or cancelled, and only the primary GM (not Co-GMs) can edit game settings.

## Basic Settings

- **Title** — The name of the game (required)
- **Description** — Shown to all players browsing for games (required)
- **Genre** — Optional label such as Fantasy, Sci-Fi, Horror, or Modern
- **Maximum Players** — How many player characters the game supports (leave blank for default of 6, max: 20)

## Dates

All date fields are optional and informational — they don't automatically trigger state changes.

- **Recruitment Deadline** — When you plan to close applications
- **Start Date** — When you plan to start the game
- **End Date** — When you plan to finish the game

## Privacy and Access

**Anonymous Mode** — When enabled, players cannot see who controls which character. Character ownership and NPC status are hidden from the player view. GMs always see full ownership information. Use this when the mystery of who plays who is part of the game.

**Auto-Accept Audience Members** — When enabled, anyone who clicks **Join as Audience** is immediately added without GM review. When disabled, audience applications appear in the People tab for you to approve or reject manually.

## Conversation Settings

**Allow Group Conversations** — Enabled by default. When on, players can start private message conversations with three or more participants. When off, private messages are limited to one-on-one conversations only.

## Creating a Game

From the **Games** page, click **Create Game**. After filling in the form and submitting, you are taken directly to your new game, which starts in **Setup** state. From there, use the game actions menu (⋮) to move to **Recruitment** when you're ready to accept applications.
