# User Profiles

Every user on ActionPhase has a public profile page at `/users/[username]`.

## What's on a Profile

- **Avatar and username**
- **Bio** — The user's self-written description (supports Markdown)
- **Member since** — The date the account was created
- **Game history** — A paginated grid of games the user has participated in

Each game card in the history shows:
- Game title (links to the game)
- Game state and the user's role (GM, Co-GM, Player, Audience)
- The GM's username
- Characters the user played (hidden for anonymous games)
- Date range of the game

## Privacy

**Anonymous Mode** hides character names from profile pages while a game is in progress — the game still appears in the history with role and state, but no character details are shown. The card is labelled **Anonymous Game**.

When a game completes, anonymous mode is automatically lifted and character details become visible on profile pages like any other game.

## Finding Someone's Profile

Links to user profiles appear throughout the site: usernames in the Common Room and private messages, player names on character cards in the People tab, and the character profile page (which links to the player's user profile).

You can also navigate directly to `/users/[username]`.
