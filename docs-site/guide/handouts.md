# Handouts

Handouts are documents the GM can share with players — reference materials, lore, rules summaries, maps descriptions, or anything else relevant to the game.

## Viewing Handouts

Published handouts appear in the **Handouts** section of the game. Click **View** on any handout to read the full content.

Handouts support Markdown formatting, so they can include headers, lists, tables, and other structured content.

### Updates & Comments

Each handout has an **Updates & Comments** section where the GM can post clarifications or updates. These appear below the main handout content and are visible to all players.

---

## GM: Creating a Handout

Click **Create Handout** from the Handouts section. The creation form includes:

- **Title** — The name of the handout
- **Content** — The body of the handout, supports Markdown
- **Status** — Start as **Draft** to work on the handout before sharing, or set to **Published** to share immediately

## GM: Managing Handouts

From the Handouts list, each handout card shows:

- **Edit** — Opens the handout editor to change the title, content, or status
- **Publish / Unpublish** — Toggle visibility to players. Players can only see Published handouts.
- **Delete** — Permanently removes the handout (requires confirmation)

Drafts are only visible to GMs. Players see the handout list only when it contains published handouts.

## GM: Adding Comments

Open any handout and scroll to the Comments section to post a clarification or update. This is useful for answering player questions about a handout's content without editing the original.

Comments can be edited or deleted after posting.
