# User Settings

Access your settings from the navigation menu at any time. Changes take effect immediately and sync across devices.

## Profile

Update your **bio** and **avatar** from the Profile section. Your public profile page is linked from here.

## Appearance

Choose a display theme:

- **Light** — Default light mode
- **Dark** — Dark mode
- **High Contrast** — Light mode with increased contrast
- **High Contrast Dark** — Dark mode with increased contrast
- **Colorblind-Friendly** — Adjusted color palette
- **System** — Follows your device's system preference

## Reading

The **Comment Read Tracking** setting controls how new comments are highlighted when you return to a page.

- **Automatic** — Comments added since your last visit are highlighted automatically.
- **Manual** — You mark individual comments as read yourself. Read comments fade out.
