# Character Public Profile

Every character has a public profile page at `/characters/:id`. You can reach it by clicking a character's name or avatar anywhere in the game.

## What's on the Page

**Header** — The character's avatar, name, type (Player Character or NPC), status, and the username of the player who controls them.

**Activity stats** — A count of the character's public messages and (if you have access) private messages.

**Activity feed** — A chronological list of all public posts and comments made by this character across all games, with the most recent first. Clicking any entry navigates to that message in context.

The feed loads more entries automatically as you scroll down.

## What's Visible

The activity feed shows only **public** Common Room posts and comments. Private messages and action submissions are not included.

The player's username is always shown on this page, even in games with **Anonymous Mode** enabled. Anonymous mode only hides ownership within the game view itself.
